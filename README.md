# DARSHAN-OS Portfolio

A modern, autonomous portfolio website with an OS-metaphor interface.

## Tech Stack
- **Frontend**: React, Tailwind CSS v4, Framer Motion (animations)
- **Backend**: Node.js, Express
- **Email**: Nodemailer (Gmail SMTP)
- **Design**: Glassmorphism, Inter Typography

## Setup
1. `npm install` in both `frontend` and `backend` folders.
2. Create `.env` in backend with `EMAIL_USER` and `EMAIL_PASS`.
3. `npm run dev` in frontend, `node server.js` in backend.

## Features
- Desktop Interface
- Voice Control (Command Prompt style)
- Projects, Skills, Experience Showcases
- Admin Panel
- Real Contact Form
